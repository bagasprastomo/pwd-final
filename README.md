# Sistem Informasi Kepegawaian

Proyek ini adalah tugas kelompok untuk mata kuliah **Pemrograman Web Dasar**. Website ini dirancang sebagai purwarupa Sistem Informasi Kepegawaian yang dibangun menggunakan teknologi fundamental web.

## Deskripsi Proyek
Website ini dikembangkan dengan fokus pada struktur informasi dan tampilan antarmuka yang bersih, hanya menggunakan:
* **HTML5**: Untuk penataan struktur konten.
* **CSS3**: Untuk desain layout, tipografi, dan estetika visual.
